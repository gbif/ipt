  -------------------------
   IPT INSTALLATION README

   Version 0.x-SNAPSHOT
  -------------------------

To get started, please complete the following steps:

1. Download and install a MySQL 5.x database from 
   http://dev.mysql.com/downloads/mysql/5.0.html#downloads.

2. Adjust JDBC settings in pom.xml
   a) the settings at the very bottom are for testing only. 
      Each time you start the test instance (=default) any previously existing database will get erased
   b) production settings can be found around line 640 in the profile "prod" section 

3. Run "mvn" and view the application at http://localhost:8080.
   b) to start the production instance run "mvn -P prod"
